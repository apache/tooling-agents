# repos

Per-repository agent work, organized by target repository and area of focus.

Each subdirectory corresponds to an Apache repository that agents are being developed for. Within each, subdirectories cover specific areas (e.g., ASVS compliance, dependency scanning, commit review).

| Repository | Area | Description |
|------------|------|-------------|
